# Report Scaffold

This folder contains the written assets for the final professor demo and thesis-style report.

Recommended order:

1. `architecture.md`
2. `experiment_procedure.md`
3. `framework_mapping.md`
4. `presentation_notes.md`
5. `screenshot_checklist.md`

The generated analysis outputs under `results/` are intended to be copied into the report as figures and tables.
